## Descripción
Este repositorio contiene la investigación práctica sobre herramientas de documentación.

## Criterios de Evaluación

### A) Identificación de Herramientas
He seleccionado:
* **PHPDocumentor** (PHP): Estándar para el backend.
* **JSDoc** (JavaScript): Estándar para el frontend.

### B) Documentación de Componentes
El código fuente en `/src` está comentado siguiendo los estándares, incluyendo etiquetas de autor (`@author Cesar`), parámetros y retornos.

### C) Formatos
1. **HTML:** Web generada automáticamente en la carpeta `/docs`.
2. **Markdown:** Este archivo y la Wiki del repositorio.
3. **Código:** Comentarios técnicos en los archivos `.js` y `.php`.

### D) Control de Versiones
Todo el historial de cambios y la evolución del proyecto están registrados en este repositorio de GitHub.
